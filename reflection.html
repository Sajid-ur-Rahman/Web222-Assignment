Responsive design means adjust the contents according to different screen sizes. These contents can be anything, for example, software, video, game, webpage etc. Whatever screen size is used, the content should adapt the environment and run without causing any error.


Responsive design is highly used in web technologies. In real world, there are many kinds of devices with different screen sizes. From 100px to 1000px or more, any size is possible. Now, when a web designer is designing their website, they have to make sure that their website runs smoothly in every device. Because we don't know what user will use to browse a website. A user may use mobile phone, laptop, tab etc. Whatever, device a user use, they should be able to browse the website without any difficulty. To accommodate this facility, almost every website today use responsive design.


From my course I have learned that, I can build responsive website using @media rule of CSS. We just have to specify the maximum and minimum length of a targeted screen, and then have to specify how will the webpage behave within this range. For example,

@media (min-width: 992px) and (max-width: 1300px)
{
  p{ color:yellow;}
}

This @media rule is targeted for typical laptop's screen or any device screen that falls within this range. For simplicity, I have specified the color of all the p tag to be yellow. If user opens my webpage using his/her laptop, he will see all the p tag in yellow.

@media (min-width: 576px) and (max-width: 799.98px)
{
  p{ color:red;}
}

This @media rule is targeted for typical smartphone's screen or any device screen that falls within this range. If user opens my webpage using his/her smartphone, he will see all the p tag in red.
This is just a basic demonstration of responsive design in web technologies.
